# Airthings


Custom component for using Airthings wave in Home Assistant

## Install
Copy the files to the custom_components folder in Home Assistant config.

In configuration.yaml:

```
    sensor:
      - platform: airthings_wave
        scan_interval: 1150
        mac: a4:da:32:14:95:4a
        plus: True
```
