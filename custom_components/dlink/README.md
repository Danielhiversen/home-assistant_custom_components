# Dlink


Modified component for using Dlink in Home Assistant

## Install
Copy the files to the custom_components folder in Home Assistant config.

In configuration.yaml:

```
    sensor:
      - platform: dlink
        name: name
        host: 192.168.0.3
        username: admin
        password: dlink_pswd
```
